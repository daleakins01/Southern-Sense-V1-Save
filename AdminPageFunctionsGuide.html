Admin Panel Functions Guide

Overview

This guide explains the functions of the Admin Panel. The panel is connected directly to the Firestore database. All changes are live and permanent.

Products Tab

Add New Product

Click the "Add New Product" button to open the form. Fill in all fields. The imageUrl must match a file name in the /src/ folder (e.g., my-new-candle.webp). The longDescription field can accept basic HTML like <p> tags.

Edit Product

Click the "Edit" button on any product row. The form will be pre-filled with that product's data. You can change any value and click "Save Product" to update it.

Delete Product

After clicking "Edit", a "Delete Product" button will appear. This will permanently delete the product from the database. This action cannot be undone.

Orders Tab

This tab displays all customer orders, sorted by the most recent. You can view the Order ID, date, customer name/email, total, and current status.

(Future Feature): Clicking "View" will open a detailed modal to update the order status (e.g., from "processing" to "shipped") and see the items purchased.

Reviews Tab

(Pillar 2 - Not Implemented): This panel will allow you to approve or deny new customer reviews before they appear on the product pages.

Site Content Tab

(Pillar 2 - Not Implemented): This panel will allow you to edit text on pages like "About Us" or "Refill Program" without needing to deploy new code.
